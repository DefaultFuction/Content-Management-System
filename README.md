## 📋 Overview

Jeson CMS is a clean, lightweight Content Management System built with native PHP and MySQL. Designed for small websites and personal bloggers, it provides essential tools to manage website content efficiently.

---

## ✨ Features

| Module | Capabilities |
|--------|--------------|
| 🔐 **Authentication** | Secure admin login system with session management |
| 📊 **Dashboard** | Website statistics and quick overview |
| 📝 **Article Management** | Publish, edit, delete, and view articles |
| 🏷️ **Category Management** | Create and manage article categories |
| 💬 **Comment System** | Moderate and manage reader comments |
| 📄 **Page Management** | Manage independent pages like About Us, Contact |
| 🖼️ **Media Library** | Upload and manage images, files and resources |
| ⚙️ **Site Settings** | Configure site title, keywords, description and more |
| 👤 **User Management** | Manage admin accounts and permissions |
