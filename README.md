## 📋 Overview

Jeson CRM is a clean, minimalist Customer Relationship Management system built with native PHP and MySQL. Designed for small businesses and startups, it provides essential tools to manage customers, products, and orders efficiently.

---

## ✨ Features

| Module | Capabilities |
|--------|--------------|
| 🔐 **Authentication** | Secure login system with session management |
| 📊 **Dashboard** | Real-time statistics and recent activities |
| 👥 **Customer Management** | Add, edit, delete, and view customer details |
| 📦 **Product Management** | Track inventory, pricing, and product information |
| 📝 **Order Processing** | Create multi-item orders with automatic total calculation |
| 📈 **Stock Control** | Automatic stock updates when orders are placed |
| 🖥️ **Server Monitoring** | System status checks via command execution |
| 🌐 **API Integration** | Built-in endpoint for external API requests |
